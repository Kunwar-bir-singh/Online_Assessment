Online Assignment – Raft Labs

This repository contains the full-stack implementation for the Raft Labs assignment.

Backend: NestJS
Frontend: Next.js
Database: PostgreSQL
Real-time updates: Server-Sent Events (SSE)


Deployed Applications

Frontend:
https://online-assessment-1-7tlz.onrender.com/

Backend:
https://online-assessment-2nhb.onrender.com/


IMPORTANT NOTE

Both the frontend and backend are deployed on Render’s free tier.

Because of this:

Services may pause automatically due to inactivity

Initial requests may take a few seconds to respond (cold start)

Please ensure both services are running simultaneously for the application to work correctly and as expected.
